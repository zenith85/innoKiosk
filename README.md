# Kiosk Project

Web-based interactive kiosk application.

## Stack
- Node.js
- Express
- Frontend: HTML / JavaScript
- Backend: REST API

## Project Structure
kiosk/
├── public/        # Frontend UI
├── server/        # Backend API
├── package.json
├── package-lock.json
└── README.md

## Setup
npm install
npm run dev

## Usage
- Open: http://localhost:3000
- Optimized for touch-based kiosk displays

## Notes
- node_modules/ is ignored
- data/ is runtime-only and not versioned
